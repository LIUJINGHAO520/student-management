## 📞 常见问题解答

### Q: 为什么看不到美化效果？
A: 需要运行 `npm install` 和 `npm run dev`。之后在浏览器中打开 http://localhost:5173

### Q: 在小屏幕上表格列太多看不全？
A: 这是故意的！在 < 768px 屏幕上，次要列（课程、教师、学期）会隐藏。大屏幕才显示全部。

### Q: 能改变颜色吗？
A: 可以！所有颜色都在 CSS 最上面，易于修改。或者后续集成 CSS 变量进一步优化。

### Q: 动画会卡顿吗？
A: 不会。所有动画都使用 `transform` 和 `opacity`，由 GPU 加速，性能最优。

### Q: 是否支持暗黑模式？
A: 当前不支持，但已预留扩展空间。未来可添加。
